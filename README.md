# Apache Airflow Orchestration Project

This project demonstrates practical usage of Apache Airflow for workflow orchestration in data engineering.

It includes examples of DAG creation, task dependencies, XCom communication, Airflow Variables, and JSON-based configuration.

---

# Technologies

* Apache Airflow
* Python
* Docker
* Astro CLI
* PostgreSQL
